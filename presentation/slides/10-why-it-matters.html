The file system metaphor is not merely an interface convention‚Äîit IS the authentic organizational structure through which born-digital materials were created and managed. Our approach balances two complementary navigation paradigms in a single interface. Vertical navigation provides familiar access patterns aligned with archival provenance. Horizontal navigation exposes complex contextual networks. The integration addresses a fundamental limitation of conventional tree visualizations, which constrain users to single descent paths and obscure relational density. Critically, the lazy loading mechanism is ontology-agnostic, operating on any RDF graph with hierarchical relationships. Within the archival domain, the foundation on RiC-O ensures applicability to diverse typologies. The module is already being tested on the Giuseppe Albini Archive, which has different structural characteristics‚Äîverifying flexibility across varying archival natures.
